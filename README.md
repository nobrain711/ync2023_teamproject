# frontend

## 주의 사항
```
현재 eslint와 vue의 설정 안되는 단어의 충돌로 인해 제외 단어에서 예외를 설정해야지   
오류 없이 돌아 갑니다.   
.eslintrc.js에서 ignores배열에 뷰 파일명을 지정해 주시면 됩니다.   
```
